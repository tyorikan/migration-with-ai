1. オンボーディング
Google Cloud の基礎と検証用の環境構築を実施するための作業が記述されたもの。

2. データベース移行
SQL 変換と最適なスキーマ移行戦略を検証するために必要な作業。

3. アプリのモダナイズ
Gemini を中心に、AI 駆動開発を用いて、既存コードの移行やモダナイズのパスを明確にしたもの。

4. インフラ＆パイプライン
IaC（Infrastructure as Code）by Terraform と CI/CD 環境の構築を行うための手順。

5. ドキュメント作成
Gemini を活用し、設計書や移行ドキュメントの自動生成を行うための作業。

6. テスト
テストコードや自動テスト環境の構築を行うための手順、コードの手引き。
